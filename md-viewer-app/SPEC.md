## 1. 產品概述

### 1.1 名稱與定位
- **名稱：** md-viewer
- **類型：** macOS 原生桌面應用
- **定位：** 純閱讀導向的 Markdown 預覽器（零編輯功能）
- **目標用戶：** 需要快速預覽 .md 檔案、不需要編輯功能的使用者

### 1.2 核心價值
- **乾淨**：沒有編輯區，只有預覽
- **快速**：拖放即開，無需等待
- **便攜**：雙擊 .md 檔直接用 App 開啟（macOS 檔案關聯）

## 2. 功能需求

### 2.1 必須具備（MVP）

| 功能 | 說明 |
|------|------|
| **開啟檔案** | 支援 Open File（Cmd+O）與拖放 .md 檔 |
| **Markdown 渲染** | 完整 GFM 語法支援 |
| **側邊欄檔案列表** | 開啟資料夾時自動掃描所有 .md，側邊欄顯示列表 |
| **預覽區** | 主要顯示區域，不可編輯 |
| **深色模式** | 自動跟隨 macOS 系統主題 |

### 2.2 預期功能（未來）

| 功能 | 說明 | 優先級 |
|------|------|--------|
| **檔案關聯** | 雙擊 .md 用 md-viewer 開啟 | P1 |
| **書籤** | 收藏常用 .md 檔 | P2 |
| **搜尋** | 在當前檔案內關鍵字搜尋 | P2 |
| **Copy HTML** | 一鍵複製渲染後的 HTML | P3 |

## 3. UI 規格

### 3.1 視窗佈局

```
┌────────────────────────────────────────────────────────────┐
│  📄 md-viewer                              [─] [□] [×]    │
├──────────────┬─────────────────────────────────────────────┤
│              │                                             │
│  📁 側邊欄   │         Markdown 預覽區                    │
│              │                                             │
│  ▸ folder1/ │  # 標題                                    │
│    ▸ notes  │                                             │
│      readme │  內文內容...                                │
│      guide  │                                             │
│    ▸ docs   │  ```go                                    │
│              │  code block                               │
│              │  ```                                      │
│              │                                             │
│              │                                             │
├──────────────┴─────────────────────────────────────────────┤
│  📄: readme.md                              🌙 深色模式    │
└────────────────────────────────────────────────────────────┘
```

### 3.2 側邊欄
- **寬度：** 固定 250px，可透過拖曳調整（未來）
- **內容：** 資料夾樹狀結構，展開/收合
- **高亮：** 當前選中的檔案有背景色標示
- **操作：** 點擊切換預覽，右鍵顯示功能選單（未來）

### 3.3 預覽區
- **技術：** WebView（HTML/CSS 渲染）
- **最大寬度：** 800px，內容居中
- **樣式：** 參考 GitHub Markdown 樣式
- **滾動：** 獨立滾動，不連動側邊欄

### 3.4 狀態列
- **位置：** 底部，高度 24px
- **內容：** 當前檔案路徑、右上角深色模式指示

### 3.5 選單列（Menu Bar）

| 選單 | 項目 |
|------|------|
| **檔案** | 開啟檔案... (⌘O)、開啟資料夾... (⌘⇧O)、關閉 (⌘W) |
| **顯示** | 切換側邊欄 (⌘B) |
| **說明** | 關於 md-viewer |

## 4. 技術規格

### 4.1 技術棧

| 層面 | 技術選擇 | 理由 |
|------|----------|------|
| **語言** | Go 1.21+ | 主要開發語言 |
| **GUI 框架** | Fyne 2.x | 純 Go，可 build 成 .app |
| **Markdown 解析** | goldmark (yuin) | Go 生態高品質解析器，支援 GFM |
| **WebView 渲染** | Fyne WebView | 內建支援，用 HTML/CSS 渲染 Markdown |
| **構建工具** | `fyne package` CLI | 官方打包工具 |

### 4.2 依賴

```go
require (
    fyne.io/fyne/v2 v2.5.0
    github.com/yuin/goldmark v1.7.4
)
```

### 4.3 資料夾結構

```
md-viewer-app/
├── main.go              ← 入口
├── go.mod
├── go.sum
├── ui/
│   ├── window.go        ← Window 初始化
│   ├── sidebar.go        ← 側邊欄元件
│   └── preview.go        ← 預覽區 WebView
├── core/
│   ├── renderer.go       ← Markdown → HTML
│   └── file_tree.go      ← 資料夾掃描與樹狀結構
└── assets/
    └── styles.css        ← Markdown 樣式
```

## 5. 驗收標準

### 5.1 MVP 驗收

- [ ] 雙擊 md-viewer.app 啟動，顯示空白 Window
- [ ] Cmd+O 可選擇 .md 檔案並正確渲染
- [ ] 拖放 .md 到 App 視窗可開啟預覽
- [ ] 側邊欄可掃描並列出目錄中的 .md 檔案
- [ ] 深色模式下樣式正確切換
- [ ] Cmd+W 可關閉 App
- [ ] 編譯後的 .app bundle < 30MB

### 5.2 已知限制

- 不支援編輯（設計如此，不視為 bug）
- 不支援多 Tab（未來功能）
- WebView 依賴 macOS WebKit 版本，部分 CSS 表現可能略有差異
