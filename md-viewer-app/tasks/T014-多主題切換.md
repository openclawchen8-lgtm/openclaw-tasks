## 目標

內建多套 CSS 主題，支援手動切換與 `prefers-color-scheme` 自動切換。

## 實際實作

### 5 套主題（main.go cssContent）

| 主題值 | 說明                                            |
| ------ | ----------------------------------------------- |
| auto   | 跟隨 macOS 系統設定（CSS prefers-color-scheme） |
| light  | GitHub 淺色風格                                 |
| dark   | GitHub 深色風格                                 |
| sepia  | 羊皮紙護眼風格                                  |
| nord   | Nord 極地風格                                   |

> **注意**：task 原始描述 4 主題（GitHub/夜間/學術/極簡），實際實作為 5 主題（light/dark/sepia/solarized/nord）。solarized 和 nord 為額外實作。

### CSS 變數架構

```css
:root,
[data-theme="light"] {
  /* GitHub 淺色變數 */
}
[data-theme="dark"] {
  /* GitHub 深色變數 */
}
[data-theme="sepia"] {
  /* 羊皮紙變數 */
}
[data-theme="solarized"] {
  /* Solarized 變數 */
}
[data-theme="nord"] {
  /* Nord 變數 */
}
@media (prefers-color-scheme: dark) {
  [data-theme="auto"] {
    /* 跟隨系統深色 */
  }
}
```

### 設定面板整合

- Settings Panel 有 theme select dropdown
- 變更後即時套用（JS `applyTheme()`）
- 選擇寫入 `~/.md-viewer/config.json` 持久化

### 驗收條件（達成）

- [x] 可切換 5 套主題（light/dark/sepia/solarized/nord）
- [x] 主題切換即時生效（JS 動態套用）

- [x] `prefers-color-scheme` 自動切換正常運作（auto 模式）
- [x] 選單顯示當前選中的主題
