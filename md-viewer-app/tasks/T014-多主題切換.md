## 目標

內建多套 CSS 樣式表，支援手動切換與 prefers-color-scheme 自動切換深淺色主題。

## 功能規格 (Technical Specs)

### 核心需求
1. **多套主題**：內建至少 4 種主題
   - GitHub（預設）
   - 夜間（Dark）
   - 學術（Academic）
   - 極簡（Minimal）
2. **自動切換**：支援 `prefers-color-scheme` 自動切換深淺色
3. **手動切換**：在選單或工具列切換主題
4. **持久化**：記住用戶選擇的主題

### 主題設計

#### GitHub（預設）
- 模擬 GitHub README 樣式
- 淺色背景、藍色鏈結、圓角表格

#### 夜間（Dark）
- 深色背景（#1e1e1e）
- 淺色文字、柔和鏈結色
- 適合夜間閱讀

#### 學術（Academic）
- 類似論文排版
- 襯線字體、正式感
- 適合技術文件

#### 極簡（Minimal）
- 最小化樣式
- 純黑白、無多餘裝飾
- 適合專注閱讀

### 技術實作

**CSS 變數架構**：
```css
:root {
  --bg-color: #ffffff;
  --text-color: #24292e;
  --link-color: #0366d6;
  --code-bg: #f6f8fa;
  /* ... */
}

[data-theme="dark"] {
  --bg-color: #1e1e1e;
  --text-color: #d4d4d4;
  /* ... */
}
```

**自動切換**：
```css
@media (prefers-color-scheme: dark) {
  :root:not([data-theme]) {
    --bg-color: #1e1e1e;
    /* ... */
  }
}
```

## 使用說明 (User Guide)

在工具列或選單切換「GitHub」、「夜間」、「學術」等風格，讓閱讀氛圍符合當前心情。
