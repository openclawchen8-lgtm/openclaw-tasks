## 實際實作狀態

### 已實作的快捷鍵

| 快捷鍵         | 功能      | 實作方式                         |
| -------------- | --------- | -------------------------------- |
| ⌘+ / ⌘=        | 放大      | JS `window.zoomIn()`             |
| ⌘-             | 縮小      | JS `window.zoomOut()`            |
| ⌘0             | 重置 100% | JS `window.zoomReset()`          |
| ⇧⌘R            | 重置 100% | JS `window.zoomReset()`          |
| Ctrl+滾輪      | 縮放      | JS `wheel` event handler         |
| Trackpad Pinch | 縮放      | JS `gesturechange` event handler |

### 縮放範圍

- 最小：50%（`minZoom = 0.5`）
- 最大：200%（`maxZoom = 2.0`）
- 靈敏度：1-10（動態 step，設定面板可調整）
