## 目標

在 NSMenu → File 選單中新增「最近開啟的檔案」子選單，列出最近開啟的 .md 檔案（最多 10 筆）。

## 實作方式

### config.json 新增欄位

```json
{
  "recentFiles": [
    "/Users/claw/Documents/readme.md",
    "/Users/claw/Notes/daily.md"
  ]
}
```

### 記錄開啟的檔案

每次成功開啟檔案時更新（不重複、保持在 10 筆以內）：

```go
func addRecentFile(path string) {
    // 移除已存在的（避免重複）
    newList := []string{}
    for _, p := range currentConfig.RecentFiles {
        if p != path { newList = append(newList, p) }
    }
    // 插入到最前面
    currentConfig.RecentFiles = append([]string{path}, newList...)
    // 限制 10 筆
    if len(currentConfig.RecentFiles) > 10 {
        currentConfig.RecentFiles = currentConfig.RecentFiles[:10]
    }
    saveConfig()
}
```

### NSMenu 動態更新

每次選單顯示前（`menuWillDisplay`）動態重建子選單：

```objc
- (void)menuWillDisplay:(NSMenu *)menu {
    // 清除舊的 Recent 項目
    // 根據 recentFiles 動態加入 NSMenuItem
}
```

### Go 端處理

點擊「最近開啟」時，menu callback 收到 ID，直接讀取 `currentConfig.RecentFiles[idx]` 並呼叫 `loadFile`。

## 驗收條件

- [ ] NSMenu → File 出現「最近開啟的檔案」子選單
- [ ] 每次成功開啟 .md 自動加入列表
- [ ] 超過 10 筆自動移除最舊的項目
- [ ] 點擊子選單項目能正確開啟該檔案
- [ ] 路徑失效時（檔案已刪除）顯示為灰色並可點擊移除
