## 目標

讓 ⌘F 正確觸發全螢幕切換。

## 根因

1. `MDAppDelegate.targetWindow` 永遠是 nil → `[nil toggleFullScreen:]` 無效
2. `MenuFullscreen` ID 與 `MenuAboutHelp` 都是 10，衝突

## 實際修改

### `menu.m`

```objc
- (void)menuFullscreen {
    NSWindow *window = [NSApp keyWindow];
    if (window && [window respondsToSelector:@selector(toggleFullScreen:)]) {
        [window toggleFullScreen:nil];
    }
}
```

### `menu.go`

```go
const (
    MenuToggleSidebar = 9
    MenuAboutHelp     = 10
    MenuFullscreen    = 11  // 修正衝突
)
```

- [x] ⌘F 正確進入/退出全螢幕
- [x] Go menu ID 無衝突
