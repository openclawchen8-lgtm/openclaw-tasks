## 目標

整合 swift-markdown 解析 AST，實現高效 Markdown 渲染。

## 實際實作

### Swift-markdown 核心（Sources/MarkdownEngine/Engine.swift）

- 使用 `swift-markdown` 解析 Markdown AST
- 自定義 HTML Renderer 輸出 GitHub Flavored Markdown HTML
- 支援：標題、段落、程式碼區塊（含語言識別）、表格、blockquote、任務清單、連結自動轉換

### 異步渲染

- Go 端 `renderer.Render(md)` 為同步呼叫（Swift 解析速度足夠快）
- WebView 透過 `wv.SetHtml()` 非同步更新，不阻塞 main goroutine
- fsnotify 觸發 reload 時使用 `wv.Dispatch()` 確保在 UI 執行緒執行

### GFM 支援

- 表格：`goldmark/extension/table` → Swift 實作
- 刪除線：Swift-markdown 原生支援
- 任務清單：`[ ]` / `[x]` → `<input type="checkbox">`
- 自動連結：URL 自動轉為 `<a href>`

### 驗收條件（達成）

- [x] Swift-markdown 正確解析 GFM 語法
- [x] UI 不卡死（WebView 非同步渲染）
- [x] GFM 語法全部正確渲染
- [x] `swift build -c release` 成功產生 libMarkdownEngine.dylib
