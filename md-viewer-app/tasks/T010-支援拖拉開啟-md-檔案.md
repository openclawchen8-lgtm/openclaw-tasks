## 目標

讓 md-viewer-webview 支援從 Finder 拖拉 .md 檔案到 app 視窗中開啟。

## 實作方式

**實際方案**：WebView HTML5 Drag & Drop + Go 檔案處理
- HTML Drop Zone UI + CSS 樣式（藍色虛線框）
- JS 監聽 dragenter/dragleave/dragover/drop 事件
- 使用 `file.path` 取得檔案路徑（非標準但 WebKit 支援）
- Go 端 `openFileByPath` function 處理實際檔案讀取與渲染

## 修改的檔案

- `main.go`:
  - 新增 `openFileByPath` function
  - 新增 CSS drop zone 樣式
  - 新增 HTML drop zone 元件
  - 新增 JS drag & drop 事件處理
  - 修正 command-line 參數處理的順序問題

## 驗收標準

- [x] 可以從 Finder 拖拉 .md 到 app 開啟
- [x] 拖拉時有視覺提示（藍色虛線框）
- [x] 非 .md 檔案顯示適當提示
- [x] 不影響現有 ⌘O 開檔功能
