## 目標

讓 md-viewer-webview 支援從 Finder 拖拉 .md 檔案到 app 視窗中開啟。

## 實際實作

**HTML5 Drag & Drop + FileReader API（JS 層）+ Go Binding**

- WKWebView 內部可接收 HTML5 `drop` 事件
- `FileReader.readAsText()` 讀取檔案內容，透過 `window.loadFileContent(name, content)` Go binding 傳遞
- 不需要檔案路徑，直接傳內容字串
- Native overlay（`DragDropWindowHelper`）保留作為 intra-app drag 的 fallback

## 修改的檔案

| 檔案 | 變更 |
|------|------|
| `main.go` | `loadFileContent` binding、CSS drop zone、HTML drop zone 元件、JS drag 事件 |
| `dragdrop.go/.m` | CGO bridge + ObjC DragDropWindowHelper overlay |

## 驗收標準（達成）
- [x] 可以從 Finder 拖拉 .md 到 app 開啟
- [x] 拖拉時有視覺提示（藍色虛線框）
- [x] 非 .md 檔案顯示錯誤提示
- [x] 不影響現有 ⌘O 開檔功能
