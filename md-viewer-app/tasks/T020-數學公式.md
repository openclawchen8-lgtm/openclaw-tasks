## 目標

整合 MathJax 或 KaTeX，支援 LaTeX 數學公式渲染。

## 功能規格 (Technical Specs)

### 核心需求
1. **LaTeX 支援**：處理 `$...$`（行內）與 `$$...$$`（區塊）語法
2. **渲染引擎**：使用 MathJax 或 KaTeX
3. **效能**：快速渲染，不卡頓
4. **主題整合**：公式樣式與主題一致

### 技術實作

**方案 A：MathJax（推薦）**
- 功能完整，支援所有 LaTeX 語法
- 檔案較大（~1MB）
- CDN 或本地載入

**方案 B：KaTeX**
- 更快（無需 JavaScript 執行）
- 檔案較小
- 部分進階語法不支援

### 推薦方案

**使用 MathJax**：

```html
<!-- template.html -->
<script>
window.MathJax = {
    tex: {
        inlineMath: [['$', '$'], ['\\(', '\\)']],
        displayMath: [['$$', '$$'], ['\\[', '\\]']]
    },
    svg: {
        fontCache: 'global'
    }
};
</script>
<script id="MathJax-script" async
    src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-svg.js">
</script>
```

**或使用 KaTeX**：
```html
<link rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/katex@0.16.9/dist/katex.min.css">
<script defer
    src="https://cdn.jsdelivr.net/npm/katex@0.16.9/dist/katex.min.js">
</script>
<script defer
    src="https://cdn.jsdelivr.net/npm/katex@0.16.9/dist/contrib/auto-render.min.js">
</script>
<script>
    document.addEventListener("DOMContentLoaded", function() {
        renderMathInElement(document.body, {
            delimiters: [
                {left: '$$', right: '$$', display: true},
                {left: '$', right: '$', display: false}
            ]
        });
    });
</script>
```

## 使用說明 (User Guide)

支援撰寫科學公式與數學符號。例如輸入 $\sum_{i=1}^{n} i$，預覽器會顯示正確的加總符號。
