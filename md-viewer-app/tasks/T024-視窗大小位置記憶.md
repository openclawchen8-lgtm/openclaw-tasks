## 目標

紀錄使用者調整過的視窗位置與大小，下次開啟自動還原。

## 現況

每次啟動視窗預設出現在上次 macOS 記錄的位置（非 md-viewer 自行管理）。

## 實作方式

### config.json 新增欄位

```json
{
  "windowFrame": { "x": 100, "y": 100, "width": 1200, "height": 800 }
}
```

### 讀取位置

啟動時（在 `wv.SetSize` 之前）：

```go
if currentConfig.WindowFrame != (ConfigWindowFrame{}) {
    frame := currentConfig.WindowFrame
    wv.SetSize(int(frame.Width), int(frame.Height), webview.HintNone)
    // CGO: [window setFrame:NSRect animate:NO]
}
```

### 儲存位置

使用 Go 的定時或事件觸發儲存：

```go
// 方案A：定時（每 30 秒寫入一次，減少 IO）
// 方案B：app deactivate 時儲存（推薦）

func saveWindowFrame() {
    if windowPtr != nil {
        frame := getWindowFrame(windowPtr) // CGO
        currentConfig.WindowFrame = frame
        saveConfig()
    }
}
```

### CGO 取得 NSWindow Frame

在 `menu.m` 或 `dragdrop.m` 新增：

```objc
//export getWindowFrame
NSDictionary* getWindowFrame() {
    NSRect frame = [window frame];
    return @{@"x": @(frame.origin.x),
             @"y": @(frame.origin.y),
             @"width": @(frame.size.width),
             @"height": @(frame.size.height)};
}
```

### 全螢幕偵測

全螢幕時不應儲存 frame（全螢幕 frame 覆蓋整個螢幕），造成退出全螢幕後視窗變形。

```objc
if (![window isInFullScreenMode]) {
    // 只在全螢幕關閉時儲存
}
```

檔案記憶 目前狀態：

✅ Cmd+O 開啟檔案 → 會記住
✅ 重新開啟 app → 自動開啟上次檔案
⚠️ Finder 拖入到 WebView 視窗 → 被 WKWebView 擋住，無法實現
