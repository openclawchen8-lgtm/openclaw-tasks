# T018 - 置頂小窗模式

- **Status**: skip
- **Type**: Feature
- **Assignee**: —
- **Created**: 2026-04-24
- **Updated**: 2026-04-28

## 目標（原始）

實作「總在最上層」模式，讓預覽視窗可以置頂，並支援透明度調整。

## 判定：Skip

md-viewer-webview 從未實作置頂模式。webview_go 的 `Window()` 回傳 NSWindow pointer，但 CGO overlay 目前只用於 drag & drop，未擴充至 window level 控制。

### 實作缺口
- ❌ 無 NSWindow setLevel 呼叫
- ❌ 無透明度調整
- ❌ 無小窗模式

### 若日後要實作
需在 `menu.m` / `dragdrop.m` 中新增：
```objc
// 置頂
[window setLevel:NSFloatingWindowLevel];
// 取消置頂
[window setLevel:NSNormalWindowLevel];
// 透明度
[window setAlphaValue:0.8];
```
然後透過 Go binding 控制 `setWindowLevel(level int)` / `setWindowAlpha(alpha float64)`。
