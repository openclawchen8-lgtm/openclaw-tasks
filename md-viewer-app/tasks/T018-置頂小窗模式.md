## 目標

實作「總在最上層」模式，讓預覽視窗可以置頂，並支援透明度調整。

## 功能規格 (Technical Specs)

### 核心需求
1. **視窗置頂**：設定視窗 level 為 floating
2. **透明度調整**：可調整視窗透明度（50%-100%）
3. **小窗模式**：縮小視窗尺寸，適合放在螢幕角落
4. **快速切換**：快捷鍵或選單切換置頂模式

### 技術實作

**視窗置頂（macOS）**：
```go
// 使用 Cocoa API
// #cgo CFLAGS: -x objective-c
// #import <Cocoa/Cocoa.h>
//
// void setWindowFloating(void *window) {
//     NSWindow *win = (NSWindow *)window;
//     [win setLevel:NSFloatingWindowLevel];
// }
```

**或使用 webview 的 setWindowFlag**：
```go
// webview/webview_go 可能不直接支援
// 需要透過 CGO 呼叫 Cocoa API
```

**透明度調整**：
```go
// 設定視窗 alpha 值
// macOS: [window setAlphaValue:0.8]
```

**小窗尺寸**：
- 正常模式：800x600（或用戶設定）
- 小窗模式：400x300
- 位置：螢幕右下角

## 使用說明 (User Guide)

開啟「總在最上層」模式，將預覽窗縮小放在螢幕角落，邊寫代碼邊參考文檔，不被其他視窗遮擋。

## 子任務

### T018-A — 視窗置頂實作
- 使用 CGO 呼叫 Cocoa API
- 設定 `NSFloatingWindowLevel`
- 切換回正常 level

### T018-B — 透明度調整
- 建立透明度滑桿
- 範圍：50%-100%
- 即時更新視窗透明度

### T018-C — 小窗模式
- 縮小視窗尺寸
- 移動到螢幕角落
- 記住小窗位置

### T018-D — 選單與快捷鍵
- 選單新增「置頂模式」選項
- 快捷鍵：⌘T（或自訂）
- 切換時顯示狀態

### T018-E — 狀態持久化
- 記住置頂模式設定
- 重開 app 後恢復
- 記住小窗位置與尺寸

## 產出

- `ui/window_level.go` 新增（Cocoa API 呼叫）
- `main.go` 更新（置頂模式邏輯）
- `~/.md-viewer/config.json` 更新

- 可能需要處理跨平台差異（Windows/Linux）
- 若 CGO 太複雜，可考慮使用 AppleScript
