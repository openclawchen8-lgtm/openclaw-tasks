# T005 - 實作深色模式支援

## 目標

WebView 預覽區支援 macOS 深色模式，並提供多套主題切換

## 實際實作

### CSS 變數架構（main.go 內嵌 cssContent）

```css
:root,
[data-theme="light"] {
  --color-canvas-default: #ffffff;
  --color-fg-default: #1f2328;
  --color-border-default: #d0d7de;
  /* ... */
}
[data-theme="dark"] {
  --color-canvas-default: #0d1117;
  --color-fg-default: #e6edf3;
  /* ... */
}
[data-theme="sepia"] {
  /* ... */
}
[data-theme="solarized"] {
  /* ... */
}
[data-theme="nord"] {
  /* ... */
}
@media (prefers-color-scheme: dark) {
  [data-theme="auto"] {
    /* 深色變數 */
  }
}
```

### 6 套主題

| 主題      | 說明                |
| --------- | ------------------- |
| auto      | 跟隨 macOS 系統設定 |
| light     | GitHub 淺色風格     |
| dark      | GitHub 深色風格     |
| sepia     | 羊皮紙護眼風格      |
| solarized | Solarized 深色      |
| nord      | Nord 極地風格       |

### JS 動態切換

```javascript
window.applyTheme = function (theme) {
  document.body.setAttribute("data-theme", theme || "auto");
  // 同步切換 highlight.js 主題
};
```

### 設定面板整合

- Settings Panel 有 theme selector dropdown
- 變更後即時套用（不重啟）
- 選擇寫入 `~/.md-viewer/config.json` 持久化

### 驗收條件（達成）

- [x] WebView 注入 CSS 根據 `prefers-color-scheme` 自動切換（auto 模式）
- [x] 6 套主題可手動切換，即時響應（不重啟 App）
- [x] 深色/淺色模式樣式均完整（標題、程式碼區塊、連結、表格、blockquote 等）
- [x] 設定面板顯示當前主題
- [x] 重開 app 後保持上次選擇的主題
