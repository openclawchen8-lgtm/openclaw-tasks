## 目標

定義 config.json 的完整結構與預設值，並與 JS config 注入整合。

## Config 結構

```json
{
  "zoomSensitivity": 2,
  "theme": "auto"
}
```

### zoomSensitivity（縮放靈敏度）
- 整數 1-3，對應：
  - 1（低）：step = 0.05
  - 2（中，預設）：step = 0.10
  - 3（高）：step = 0.20

### theme（外觀主題）
- 字串：auto / light / dark
- auto：跟隨系統
- light：固定淺色
- dark：固定深色

## 修改的檔案

### `config.go`
- `Config` struct：
```go
type Config struct {
    ZoomSensitivity int    `json:"zoomSensitivity"`
    Theme           string `json:"theme"`
}
```
- `DefaultConfig()`：回傳預設值
- `ConfigToJS()`：回傳 JS 注入用的 `window.config = {...}` 字串

### `main.go`
- 啟動時注入：`wv.Eval(configToJS())`

## 驗收標準
- [x] 預設值正確（zoomSensitivity=2, theme=auto）
- [x] JS `window.config` 正確設定
- [x] Settings panel 讀取時顯示正確目前值
