## 目標

定義 config.json 的完整結構與預設值，並與 JS config 注入整合。

## Config 結構（實際）

```json
{
  "zoomSensitivity": 5,
  "theme": "auto",
  "zoomLevel": 1.0,
  "fontFamily": "-apple-system, BlinkMacSystemFont, 'Segoe UI', Helvetica, Arial, sans-serif",
  "fontSize": 16,
  "language": "zhTW",
  "showLineNumbers": false
}
```

### zoomSensitivity（縮放靈敏度）

- 整數 1-10，預設 5
- 對應 step = 0.02 × level

### theme（外觀主題）

- auto / light / dark / sepia / solarized / nord

### language（語言）

- zhTW（繁中）/ zhCN（簡中）/ en / ja / ko

## 實際修改的檔案

### `config.go`

```go
type Config struct {
    ZoomSensitivity int     `json:"zoomSensitivity"`
    Theme           string  `json:"theme"`
    ZoomLevel       float64 `json:"zoomLevel"`
    FontFamily      string  `json:"fontFamily"`
    FontSize        int     `json:"fontSize"`
    Language        string  `json:"language"`
    ShowLineNumbers bool    `json:"showLineNumbers"`
}

func ConfigToJS() string {
    return fmt.Sprintf(`window.mdConfig = {zoomSensitivity: %d, theme: %q, zoomLevel: %f, fontFamily: %q, fontSize: %d, language: %q, showLineNumbers: %t};`,
        currentConfig.ZoomSensitivity, currentConfig.Theme, currentConfig.ZoomLevel,
        currentConfig.FontFamily, currentConfig.FontSize, currentConfig.Language, currentConfig.ShowLineNumbers)
}
```

### `main.go`

- 啟動時注入：`<script>window._initConfig=window.mdConfig;</script>`（在 `<head>` 關閉前）
- IIFE init 時讀取 `_initConfig` 或 `mdConfig`
