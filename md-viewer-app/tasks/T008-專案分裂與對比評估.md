## 對比結果摘要

| 項目 | webview | fyne | 備註 |
|------|---------|------|------|
| 二進位大小 | 5.1 MB | 33 MB | webview 勝 |
| 啟動速度 | 快 | 中等 | webview 勝 |
| 渲染品質 | ★★★★★ | ★★★☆☆ | webview 勝（WebKit） |
| 縮放功能 | ✅ CSS zoom | ✅ Window resize | 皆完成 |
| 原生選單 | ✅ ⌘O/⌘W/⌘Q | ❌ 僅快捷鍵 | webview 勝 |
| 圖片開啟 | ✅ 正常顯示 | ⚠️ 會跳瀏覽器 | webview 勝 |
| 跨平台 | ⚠️ macOS only | ✅ 理論跨平台 | fyne 勝 |
| 依賴複雜度 | 低 | 高 | webview 勝 |

## 結論

**推薦：webview 版本**（md-viewer-webview）

理由：
1. 體積小 6 倍（5MB vs 33MB）
2. 渲染品質更高（真正 WebKit）
3. 原生 macOS 體驗（Native Menu）
4. 無 URL scheme 干擾問題
