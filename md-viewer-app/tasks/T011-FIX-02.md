## 目標

實作 HTML/JS 層的 i18n 機制，讓語言設定切換後所有 UI 文字和 menubar 同步更新。

## 問題描述

1. `window.applyLanguage` 從未定義 → 語言 select 的 change handler 呼叫它但無效果
2. HTML 內所有文字硬編碼（keyboard-hint 英文、settings label 繁中、empty-state 英文），無 i18n dictionary
3. `saveLanguage` Go bind 不觸發 `UpdateMenuLanguage()` → menubar 不會跟著切換

## 修改檔案

#### 新增 `applyLanguage` function

```js
window.applyLanguage = function(lang) {
  var t = I18N[lang] || I18N.en;
  // 更新 drop zone
  var dz = document.querySelector('.drop-zone-msg');
  if (dz) dz.textContent = t.dropMsg;
  // 更新 empty state
  var et = document.querySelector('.empty-state-title');
  if (et) et.textContent = t.emptyTitle;
  var ed = document.querySelector('.empty-state div:last-child');
  if (ed && ed.parentElement.classList.contains('empty-state')) ed.textContent = t.emptyDesc;
  // 更新 keyboard hint
  var kh = document.querySelector('.keyboard-hint');
  if (kh) kh.textContent = t.kbHint;
  // 更新 settings panel labels
  var st = document.querySelector('.settings-title');
  if (st) st.childNodes[0].textContent = t.settingsTitle + ' ';
  // ... 其餘 label
};
```

#### IIFE 結尾初始化語言

```js
// IIFE init 中，在 applyZoomSensitivity 之後加：
if (window.mdConfig && window.mdConfig.language) {
  window.applyLanguage && window.applyLanguage(window.mdConfig.language);
}
```

### `main.go` Go bind

修正 `saveLanguage` callback，加上 menu 更新：

```go
wv.Bind("saveLanguage", func(lang string) {
    if err := SetLanguage(lang); err != nil {
        fmt.Fprintln(os.Stderr, "Failed to save config:", err)
    } else {
        wv.Eval(getConfigJS())
        UpdateMenuLanguage(lang) // 新增：同步更新 NSMenu
    }
})
```

### `main.go` 啟動時

在 `SetupMenu` 之前，用 config 中的 language 初始化 menubar：

```go
UpdateMenuLanguage(currentConfig.Language) // 新增

```
