## 目標（原始）

側邊欄顯示目錄中的 .md 檔案列表，支援點擊切換預覽

## 判定：Skip

md-viewer-webview 從未實作側邊欄。最終產品定位為**純閱讀器**（Reader），而非 IDE 風格的 split-view 編輯器。

### 實作缺口

- ❌ 無側邊欄 HTML/CSS/JS
- ❌ 無目錄掃描功能
- ❌ 無檔案列表點擊切換邏輯

### 替代方案（已實作）

| 替代功能         | 說明                                      |
| ---------------- | ----------------------------------------- |
| ⌘O 開檔          | 透過 macOS 檔案對話框選取 .md             |
| 拖拉開啟         | 直接拖 .md 檔到視窗                       |
| 相對路徑跳轉     | `openFileByPath` Go binding               |
| 檔案監控自動重載 | fsnotify + debounce（外部編輯後自動更新） |

### 若日後要實作側邊欄

需重構 main.go 加入 split-view 佈局：

1. 左側（25%）：HTML `<div>` 列表 + CSS 樣式
2. 右側（75%）：現有 markdown 預覽區
3. Go 端：維護 `currentDir` 狀態，掃描目錄 .md 檔案
4. JS binding：點擊側邊欄項目 → `loadFileByPath`
