## 目標

使用 fsnotify 監控當前開啟的 .md 檔案，偵測到 write 事件時自動觸發渲染更新。

## 實際實作（main.go）

### fsnotify 整合
```go
import "github.com/fsnotify/fsnotify"

watcher, _ := fsnotify.NewWatcher()
watcher.Add(currentFile)  // 監控目前開啟的檔案
```

### 防抖機制
```go
debounceTimer = time.AfterFunc(150*time.Millisecond, func() {
    watcher.Remove(event.Name)
    watcher.Add(currentFile)  // Rename 後重新加入
    wv.Dispatch(func() { reloadFile() })
})
```

### Reload Flash 效果
```go
pendingFlash = true
loadFile(currentFile)
// → renderMD() 在<body>加上 class="reload-flash"
// → CSS animation：粉紅色閃爍邊框（1.5s）
```

### 支援的事件
| 事件 | 處理 |
|------|------|
| `fsnotify.Write` | 防抖 150ms → reload |
| `fsnotify.Create` | 防抖 150ms → reload |
| `fsnotify.Rename` | 重新 Remove + Add 檔案 → reload |

### 驗收條件（達成）
- [x] 開啟 .md 檔案後，外部編輯存檔 → 預覽自動更新
- [x] 防抖生效（150ms，多次回存只觸發一次更新）
- [x] Reload flash 動畫提供視覺回饋
- [x] 支援 Vim、Sublime、VSCode 等編輯器
