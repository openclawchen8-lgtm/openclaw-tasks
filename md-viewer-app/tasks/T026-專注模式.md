## 目標

點擊按鈕或快捷鍵，進入專注模式：淡化當前可見區域以外的段落，只保留正在閱讀的內容高亮，減少視覺干擾。

## 實作方式

### 快捷鍵

- `⌘\` 或 `⌘Shift\``：開關專注模式

### JS：IntersectionObserver 偵測可見段落

```javascript
var focusMode = false;

window.toggleFocusMode = function() {
    focusMode = !focusMode;
    if (focusMode) {
        document.body.classList.add('focus-mode');
        updateFocusedParagraph();
    } else {
        document.body.classList.remove('focus-mode');
        document.querySelectorAll('.dimmed').forEach(el => {
            el.classList.remove('dimmed');
        });
    }
};

function updateFocusedParagraph() {
    // 移除所有 dimmed
    document.querySelectorAll('.dimmed').forEach(el => el.classList.remove('dimmed'));

    // 找出目前視窗中央的段落
    var viewportCenter = window.scrollY + window.innerHeight / 2;
    var paragraphs = document.querySelectorAll('.markdown-body > *');

    paragraphs.forEach(function(el) {
        var rect = el.getBoundingClientRect();
        var elCenter = window.scrollY + rect.top + rect.height / 2;
        if (Math.abs(elCenter - viewportCenter) > window.innerHeight * 0.4) {
            el.classList.add('dimmed');
        }
    });
}
```

### 滾動時更新

```javascript
window.addEventListener('scroll', debounce(updateFocusedParagraph, 100));
```

### CSS

```css
.focus-mode .dimmed {
    opacity: 0.15;
    transition: opacity 0.3s;
}
.focus-mode * {
    opacity: 1;
    transition: opacity 0.3s;
}
```

### 設定面板

可在設定中開關專注模式按鈕是否顯示。
