## 交付物

### 程式碼（index.ts 更新至 10.6KB）
| 新增功能 | 說明 |
|----------|------|
| `rollingType` 設定 | `"session"` (預設) / `"daily"` / `"size"` |
| 大小滾動 | 超過 `maxFileSizeMb` (預設 10MB) 自動換檔 |
| 每日滾動 | 每日 00:00 自動換檔 |
| 舊檔清理 | 啟動時刪除 `retentionDays` 天前的檔案 |

### Config Schema
```typescript
rollingType: "session" | "daily" | "size"
maxFileSizeMb: number  // 預設 10
retentionDays: number  // 預設 30
```
