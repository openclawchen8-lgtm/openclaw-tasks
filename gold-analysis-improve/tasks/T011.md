# T011 - gold_bot_history.py 月份 URL 支援

- **Type**: Feature
- **Assignee**: 寶寶
- **Parent**: gold-analysis-improve
- **Created**: 2026-04-19

## 需求背景

台灣銀行提供三種 URL pattern：
| URL | 用途 |
|-----|------|
| `https://rate.bot.com.tw/gold/chart/year/TWD` | 滾動一年（約 244 筆） |
| `https://rate.bot.com.tw/gold/chart/2025-01/TWD` | 指定年月（如補 2025-01 資料） |
| `https://rate.bot.com.tw/gold/chart/day/TWD` | 最近一個營業日 |

現有腳本只支援 year pattern，需擴充支援月份指定與當日查詢。

## 功能需求

- [x] 新增 `--month YYYY-MM` flag：抓取指定月份的歷史資料
- [x] 新增 `--day` flag：抓取最近一個營業日的資料
- [x] 原有 `--dry-run`、`--stats`、`--force`、`--init` 維持不變
- [x] 所有 URL 共用同一套解析邏輯（fetch_history 函式）

## 實作計畫

1. 建立 `fetch_month_history(year_month: str)` 函式，URL = `chart/{YYYY-MM}/TWD`
2. 建立 `fetch_day_history()` 函式，URL = `chart/day/TWD`
3. 更新 argparse：新增 `--month` 和 `--day` 參數
4. 處理參數衝突：`--month` 和 `--day` 不可同時使用
5. 測試：分别用 --month 和 --day 測試抓取
6. 更新 howto 文件
