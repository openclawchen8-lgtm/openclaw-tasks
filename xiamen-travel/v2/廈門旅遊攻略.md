## 小三通交通

### 方式一：套票（推薦，較輕鬆）

透過旅行社或淘寶購買「廈門來回套票」，含以下完整服務：

| 項目 | 說明 |
|------|------|
| 台灣本島 → 金門尚義機場 | 立榮/華信國內線機票 |
| 金門機場 → 水頭碼頭 | 接駁巴士 + 行李直送 |
| 金門水頭 → 廈門五通碼頭 | 船票（約30分鐘）|
| **KKday 套票（單程）** | 3,600–3,800 TWD（含台灣本島→金門機票+接送+船票）|
| **KKday 套票（來回）** | 約 7,200–7,600 TWD |
| **自行分段（來回船票）** | 約 260–360 CNY（不含機票）|

> 📌 票價來源：KKday 產品 143059（2026-04-19 查詢）
> 📌 提前在旅行社/淘寶訂購，節日期間需提前5天以上

### 方式二：自行分段訂（較便宜，需多點安排）

| 航段 | 方式 | 費用 |
|------|------|------|
| 台灣本島 → 金門 | 立榮/華信國內線 | 約 1,500-2,500 TWD |
| 金門水頭 → 廈門五通 | 船票（單程）| 約 130-180 CNY |
| **來回合計（不含機票）** | | 約 260-360 CNY |

| 方式 | 說明 | 費用 |

## 七天方案

### 概覽

| 天數 | 主題 | 住宿 |
|------|------|------|
| Day 1 | 抵達 + 中山路初探 | 中山路 |

| Day 5 | 園林植物園 + 沙坡尾 + 八市 | 曾厝垵 |

---

### Day 1｜抵達 · 中山路老街

**上午**：抵達廈門（小三通至五通碼頭 / 直飛至高崎機場）  
→ 入住中山路飯店

**下午**：中山路步行街閒逛 → 八市海鮮市場  
→ 嘗：黃則和花生湯 + 肉燕

**晚上**：八市海鮮市場（買現撈海鮮，附近餐館代客烹調）  
→ 住宿：中山路

---

### Day 2｜鼓浪嶼（上）

**早上**：東渡碼頭乘船（提前預訂船票）→ 鼓浪嶼  
**上午**：龍頭路 → 風琴博物館 → 菽莊花園 → 皓月園  
**午餐**：鼓浪嶼餐廳（上嶼水產）

**下午**：日光岩登頂 → 港仔後沙灘日落  
**晚上**：鷺江夜遊（optional）  
→ 住宿：中山路

---

### Day 3｜鼓浪嶼（下）· 文藝慢時光

**早上**：早晨漫步小巷（人少，清幽）  
**上午**：國際刻字館 → 蟲洞書店 → 筆山路（拍照聖地）

**午餐**：堂宴·老廈門私房菜

**下午**：返回厦門 → 鐵路公園 → 演武大橋觀景台（日落）  
**晚上**：曾厝垵夜市 → 沙茶麵 + 土筍凍  
→ 住宿：曾厝垵民宿

---

### Day 4｜南普陀 · 廈大 · 環島路

**早上**：南普陀寺（清晨祈福）→ 登五老峰 → 寺內素餅（伴手禮）  
**午餐**：厦大周邊

**下午**：厦門大學（需提前預約）→ 芙蓉隧道 → 上弦場  
**傍晚**：環島路騎行 → 黃厝海灘日落  
→ 住宿：曾厝垵

---

### Day 5｜園林植物園 · 沙坡尾 · 老城

**早上**：廈門園林植物園（雨林世界，07:00-09:00最佳）  
**午餐**：沙坡尾文創園區

**下午**：沙坡尾 → 頂澳仔貓街 → 白鷺洲公園  
**晚上**：中山路最後一逛（沙茶麵）  
→ 住宿：中山路

---

### Day 6｜集美學村 · 十里長堤

**早上**：地鐵1號線「海上列車」段 → 集美學村（嘉庚建築）  
**午餐**：集美大排檔

**下午**：十里長堤（日落）→ 地鐵1號線海上段（日落拍照）  
→ 住宿：曾厝垵

---

### Day 7｜最後巡禮 · 回程

**早上**：曾厝垵最後採買（伴手禮：南普陀素餅）  
**中午**：中山路最後一碗沙茶麵 → 返程

## 14天方案

### 概覽（7天精華 + 7天深度）

| 天數 | 主題 | 特色 |
|------|------|------|
| Day 1-3 | 鼓浪嶼深度（住島上1晚）| 世界文化遺產完整體驗 |
| Day 4-5 | 廈大 + 南普陀 + 環島路 | 文化+自然海岸線 |
| Day 6-7 | 老城巡禮 + 園林植物園 | 在地生活深度遊 |
| Day 8 | 八市 + 沙坡尾藝術日 | 市場文化 + 文創 |
| Day 9 | 集美學村 + 十里長堤 | 嘉庚建築 + 海堤日落 |
| Day 10 | 南靖土樓一日遊 | 世界文化遺產，距離厦門約2.5小時 |
| Day 11 | 雲水謠古鎮深度遊 | 電影取景地，千年古道 |
| Day 12 | 島內再探索：鐵路公園 + 胡里山炮台 | 歷史軍事文化 |
| Day 13 | 海上運動日（帆船/ SUP）或溫泉休閒 | 互動體驗 |
| Day 14 | 最後巡禮 + 回程 | 慢遊+採買 |

---

### Day 8｜八市 · 沙坡尾

**下午**：華僑博物院 或 白鷺洲公園

### Day 9｜集美學村（完整日）

**早上-下午**：集美學村完整探索 → 鱷園 → 陳嘉庚紀念館  
**下午-傍晚**：十里長堤日落 + 地鐵1號線海上段  
**晚上**：返回市區

---

### Day 10｜南靖土樓一日遊

**早上**：包車或跟團出發（車程約2.5-3小時）  
→ 田螺坑土樓群（「四菜一湯」）→ 裕昌樓 → 塔下村  
**下午**：返回厦門

---

### Day 11｜雲水謠古鎮

**早上**：前往雲水謠古鎮（車程約2小時）  
→ 雲水謠古鎮 + 和貴樓 + 懷遠樓  
**下午**：返回厦門

---

### Day 12｜島內歷史文化日

**早上**：鐵路公園 → 胡里山炮台  
**下午**：廈門園林植物園（補完未去區域）  
**晚上**：整理行李

---

### Day 13｜互動體驗日

**早上**：帆船出海體驗（香山/五緣灣，淘寶預訂，約150-200 CNY）  
**下午**：觀音山 或 廈門科技館  
**晚上**：曾厝垵最後一晚

---

### Day 14｜最後巡禮 · 回程

**早上**：最後一次海邊漫步 → 最後採買（素餅、餡餅）  
**下午**：返程

## 實用 Tips

### ✅ 行前準備

1. **台胞證**：確認有效期，提前申請一次簽
2. **支付**：支付寶/微信支付開通，備少量現金（路邊小吃）
3. **小三通**：建議淘寶/旅行社購買來回套票（1,300 CNY），比自行分段訂更省心
4. **預約**：厦大、鼓浪嶼船票務必提前在官方平台預訂
5. **行李**：輕裝出行，海島早晚溫差建議帶薄外套

### ⚠️ 避坑提醒

1. **鼓浪嶼船票**：只相信「廈門輪渡」公眾號，拒絕黃牛
2. **厦大門票**：免費但需實名預約，無需購買任何「代預約」服務
3. **的士/黑車**：機場、火車站避開，堅持用滴滴
4. **海鮮市場**：八市買海鮮要先問價
5. **導遊推銷**：路上拉客的私人導遊勿輕信，淘寶正規預訂

### 🎁 伴手禮推薦

- 南普陀素餅（寺內有門市）
- 鼓浪嶼餡餅（黃金香/陳記）
- 桂圓乾、閩北茶（鐵觀音/岩茶）
