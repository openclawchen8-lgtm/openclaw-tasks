| 項目 | 說明 |
|------|------|

## 交通攻略

### 🛫 外部交通

| 方式 | 說明 |
|------|------|
| **小三通** | 金門→廈門船（30分）+ 小三通巴，較實惠 |
| **直飛** | 台北→廈門高崎國際機場，約1.5小時 |
| **高鐵** | 台灣無直達，可經小三通或香港轉 |

### 🚇 市内交通

| 方式 | 說明 | 費用 |
|------|------|------|
| **地鐵** | 1/2/3/6號線覆蓋核心景點 | 2-7 CNY/次 |
| **BRT 快速公交** | 覆蓋島內主要幹線 | 1-4 CNY/次 |
| **公交** | 線路密集，支付寶可刷 | 1-2 CNY/次 |
| **共享單車** | 環島路騎行最佳選擇 | 1-3 CNY/30分 |
| **滴滴/出租** | 島內短程方便，避開高峰期 | 10-40 CNY/程 |
| **步行** | 老城區、鼓浪嶼島上，步行最舒服 | 免費 |

### ⛴️ 鼓浪嶼輪渡

- **購票**：微信搜「廈門輪渡」公眾號，提前 1-3 天預訂
- **航線**：遊客從廈鼓碼頭出發（東渡），35 CNY 往返
- **建議**：節假日務必提前 5 天預訂，平日提前 1-2 天

## 七天方案

### 概覽

| 天數 | 主題 | 住宿 |
|------|------|------|
| Day 1 | 抵達 + 中山路初探 | 中山路 |
| Day 2 | 鼓浪嶼深度遊 | 中山路 |
| Day 3 | 鼓浪嶼 + 晚間 | 中山路 |
| Day 4 | 南普陀 + 廈大 + 環島路 | 黃厝/曾厝垵 |
| Day 5 | 園林植物園 + 沙坡尾 + 八市 | 黃厝/曾厝垵 |
| Day 6 | 集美學村 + 十里長堤 | 黃厝/曾厝垵 |
| Day 7 | 最後巡禮 + 回程 | — |

---

### Day 1｜抵達 · 中山路老街

**上午**：小三通或直飛抵達廈門  
→ 入住中山路酒店（鎮海路地鐵站步行5分鐘內為佳）

**下午**：中山路步行街閒逛  
→ 騎樓建築、南洋風情老店  
→ 嘗：黃則和花生湯 + 肉燕

**晚上**：八市海鮮市場  
→ 買現撈海鮮，附近餐館代客烹調  
→ 嘗：煎蟹、白灼蝦、海蠣煎  
→ 住宿：中山路

---

### Day 2｜鼓浪嶼（上）

**早上**：06:30 出發，東渡碼頭乘船（提前預訂）  
→ 07:30 抵達鼓浪嶼

**上午**：龍頭路石板街 → 最美轉角 → 風琴博物館  
→ 菽莊花園（鋼琴博物館+四十四橋）  
→ 皓月園（鄭成功雕像）

**午餐**：鼓浪嶼餐廳（推薦：上嶼水產）

**下午**：日光岩登頂（俯瞰全島+厦門城市天際線）  
→ 港仔後沙灘日落

**晚餐**：鼓浪嶼島上或返回厦門  
→ 島上：網紅義大利餐廳 / 島下：中上路

**晚上**：鷺江夜遊（optional）  
→ 住宿：中山路

---

### Day 3｜鼓浪嶼（下）· 文藝慢時光

**早上**：再次上島（已購往返票）  
→ 早晨時段漫步小巷（人少，清幽）

**上午**：國際刻字館 → 蟲洞書店 → 番婆樓  
→ 筆山路、雞山路（遊客少，拍照聖地）

**午餐**：島上私房菜（推薦：堂宴老廈門私房菜）

**下午**：返回厦門 → 鐵路公園帶狀漫步  
→ 和平碼頭 → 鴻山隧道 → 演武大橋觀景台

**晚上**：曾厝垵夜市覓食  
→ 嘗：沙茶麵、土筍凍、燒仙草  
→ 住宿：曾厝垵民宿（海邊文創區）

---

### Day 4｜南普陀 · 廈大 · 環島路

**早上**：南普陀寺（清晨祈福，香火清幽）  
→ 登五老峰俯瞰厦大 + 城市全景  
→ 寺內素齋 / 素餅（必買伴手禮）

**午餐**：厦大周邊（咖啡一條街）

**下午**：厦門大學（需提前在官方系統預約）  
→ 芙蓉湖 → 芙蓉隧道（塗鴉牆）→ 上弦場（面海足球場）

**傍晚**：環島路騎行（黃厝海灘段）  
→ 椰風寨 → 會展中心 → 觀音山  
→ 日落時分：演武大橋觀景台 或 白城沙灘

**晚上**：環島路海鮮排檔  
→ 住宿：黃厝/曾厝垵

---

### Day 5｜園林植物園 · 沙坡尾 · 老城

**早上**：廈門園林植物園（雨林世界+多肉區）  
→ 07:00-09:00 最佳，霧氣朦朧拍照佳  
→ 門票 30 CNY，建議搭園內觀光車

**午餐**：沙坡尾文創園區（創意餐廳）

**下午**：沙坡尾藝術西區 → 老港口風情  
→ 頂澳仔貓街（小巷貓咪壁畫）  
→ 白鷺洲公園（白鷺棲息地）

**傍晚**：白鷺洲公園喝咖啡等日落

**晚上**：中山路晚餐（最後一晚中山路）  
→ 住宿：中山路（或曾厝垵）

---

### Day 6｜集美學村 · 十里長堤

**早上**：地鐵 1 號線「海上列車」段  
→ 集美學村（嘉庚建築群）  
→ 鱷園 → 陳嘉庚紀念館

**午餐**：集美大排檔（海鮮實惠）

**下午**：十里長堤海堤（日落絕佳地點）  
→ 地鐵 1 號線「海上火車」段（日落時分拍照）  
→ 杏林灣 / 園博苑（可選）

**晚上**：返回市區，曾厝垵最後一逛  
→ 住宿：曾厝垵

---

### Day 7｜最後巡禮 · 回程

**早上**：睡到自然醒，最後一次海邊漫步  
→ 黃厝海灘日出（可選）

**上午**：曾厝垵最後採買（伴手禮：南普陀素餅、餡餅）  
→ 快速移動至機場/碼頭

**中午**：根據航班/船班時間，享用最後一餐  
→ 推薦：中山路最後一碗沙茶麵

**下午**：返程

## 實用 Tips

### ✅ 行前準備

1. **台胞證**：確認有效期，提前申請一次簽
2. **支付**：支付寶/微信支付開通，備少量現金（路邊小吃）
3. **網路**：淘寶買大陸門號卡（7天/14天方案可選）或行動漫遊
4. **預約**：厦大、鼓浪嶼船票務必提前在官方平台預訂
5. **行李**：輕裝出行，海島早晚溫差建議帶薄外套

### ⚠️ 避坑提醒

1. **鼓浪嶼船票**：只相信「廈門輪渡」公眾號，拒絕黃牛
2. **厦大門票**：免費但需實名預約，無需購買任何「代預約」服務
3. **的士/黑車**：機場、火車站避開，堅持用滴滴
4. **海鮮市場**：八市買海鮮時要「先問價」，選擇透明標價店家
5. **導遊推銷**：路上拉客的私人導遊勿輕信，淘寶正規預訂

### 🎁 伴手禮推薦

- 南普陀素餅（寺內有門市）
- 鼓浪嶼餡餅（黃金香/陳記）
- 桂圓乾、閩北茶（鐵觀音/岩茶）
- 手工牛軋糖（中山路老店）
