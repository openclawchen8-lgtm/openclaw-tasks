# T002 - lifecycle.py 加回自動歸檔到 _done/

## 問題
4/9 之後歸檔機制失效。idea 檔所有 task 都 done 後，仍留在 Ideas/ 根目錄，每次掃描都會看到。

## 修法
lifecycle.py 的 `run_scan()` 結束時：
- 檢查每個 idea 檔的 done_count == total_task_count
- 若全部 done 且 Tasks/ 也無 pending → `shutil.move(idea_file, _done_dir / idea_file.name)`
- 移動後從 processed_ideas.json 移除該條目
- 加 `--no-archive` 參數可跳過歸檔

- 確認 working-issue.txt、openclaw.txt、claw-sessions-issue.txt 等全 done 的 idea 被搬到 _done/
- 確認有 pending task 的 idea 不被搬
