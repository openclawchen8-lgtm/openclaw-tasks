## 描述
檢查 GitHub CLI token 的權限範圍，確保只授予必要的權限。

## 產出
| 檔案 | 路徑 | 說明 |
|------|------|------|
| 安全評估結論 | 任務檔案內文 | 維持現狀，建議未來改用 fine-grained token |

## 評估結論
- `repo` — 完整 repo 存取
- `admin:public_key` — 管理 SSH 公鑰
- `gist` — 建立 gist
- `read:org` — 讀取組織資訊

**建議**：維持現狀，未來如有安全顧慮可改用 fine-grained token。
