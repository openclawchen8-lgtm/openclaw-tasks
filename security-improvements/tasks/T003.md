## 描述
使用 `chflags schg` 鎖定敏感檔案，防止意外修改或刪除。

## 產出
| 檔案 | 路徑 | 說明 |
|------|------|------|
| SSH 私鑰 | `~/.ssh/id_ed25519` | 已設定 schg flag |
| Shell 配置 | `~/.zshrc` | 已設定 schg flag |
| GitHub CLI | `~/.config/gh/hosts.yml` | 已設定 schg flag |

## 備註
需要管理員密碼才能執行，解鎖需要 `sudo chflags noschg <file>`。
