## 描述
將 `~/.qclaw/openclaw.json` 中的明文 API Key 改用環境變數，避免敏感資訊暴露。

## 產出
| 檔案 | 路徑 | 說明 |
|------|------|------|
| 環境變數配置 | `~/.zshrc` | 新增 OPENROUTER_API_KEY 等環境變數 |
| 配置更新 | `~/.qclaw/openclaw.json` | 已改用 `${VAR}` 語法 |

## 備註
需要重啟 shell 或 source ~/.zshrc 才能生效。
