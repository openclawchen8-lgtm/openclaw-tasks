# T005 - Cron 定時執行

## 目標
設定 Cron Job，每日定時自動抓取台股報告。

## 待辦項目
- [☐] 確認執行時間（建議：每日 09:05）
- [☐] 建立 Cron Job
- [☐] 設定 sessionKey
- [☐] 測試 Cron 執行

## 技術細節

### Cron 配置
```json
{
  "name": "永豐投顧台股報告每日抓取",
  "schedule": {
    "kind": "cron",
    "expr": "5 9 * * 1-5",
    "tz": "Asia/Taipei"
  },
  "payload": {
    "kind": "systemEvent",
    "text": "python3 /Users/claw/scripts/sinotrade_scraper.py --telegram"
  },
  "sessionTarget": "main",
  "enabled": true
}
```

### sessionKey
```
agent:main:cron:sinotrade-scraper
```

## 驗證方式
等待 Cron 自動執行，或手動觸發測試。
