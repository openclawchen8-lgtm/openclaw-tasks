# T008 - 完整報告內容讀取（需登入，暫緩）

## 目標
實作完整報告內容讀取（PDF 下載或會員全文）。

## 狀態：pending（等待用戶申請帳號）

## 原因

### 實測結果（2026-04-22）
對上銀報告（guid: 7cffdf5e-ac21-4e91-8f83-8def20c4c028）進行全面滲透測試：

| 測試項目 | 結果 | 說明 |
|----------|------|------|
| `/Content/PDF/{guid}.pdf` | ❌ HTML 登入牆 | ASP.NET Error 頁 |
| `/Article/Download/{guid}` | ❌ HTML 登入牆 | ASP.NET Error 頁 |
| `/Download/Report/{guid}` | ❌ HTML 登入牆 | ASP.NET Error 頁 |
| `/Report/Download/{guid}` | ❌ HTML 登入牆 | ASP.NET Error 頁 |
| `POST /Article/GetSubscribeListByStock` | ❌ 500 Error | 需要 auth token |
| 詳情頁文章內容 | ❌「登入會員，看更多」| 全文被截斷 |
| 「報告下載」按鈕 | ❌ 觸發登入彈窗 | Client-side modal，無下載 API |

### 根本原因
永豐投顧報告屬於付費訂閱內容，完整報告（PDF + 全文）需要：
1. 有效的會員帳號
2. 有效訂閱方案
3. 伺服器端 auth token 驗證

### 用戶情況
- 用戶**沒有永豐投顧帳號**
- 方向 A（Cookie 注入）：無法執行（無帳號、無 Cookie 可匯出）
- 方向 B（公開替代格式）：已確認不存在

## 未來可行方向

### 方向 1：取得帳號後（用戶自選）
1. 用戶至永豐投顧官網註冊/訂閱
2. 提供帳號密碼或匯出瀏覽器 Cookie
3. 實作 `save_cookies.py`：Playwright 登入一次 → 匯出 JSON
4. 之後腳本讀取 Cookie 自動抓取完整報告

### 方向 2：免費替代來源
尋找提供免費台股研究報告的其他來源，例如：
- CMoney 研究報告
- 合一證券
- 公開的券商報告網站
