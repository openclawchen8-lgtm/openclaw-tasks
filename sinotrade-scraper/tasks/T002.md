## 輸出格式
```json
{
  "date": "20260422",
  "reports": [
    {
      "code": "2049",
      "name": "上銀",
      "title": "景氣初升，量價齊揚將推升獲利",
      "url": "https://scm.sinotrade.com.tw/Article/Inner/{guid}",
      "fetched_at": "2026-04-22T09:05:00+08:00"
    }
  ]
}
```

## 備註
- 詳情頁 url 直接可存取，無需登入即可看到預覽（第一段摘要）
- 預覽後被「登入會員，看更多」截斷，完整內容需登入
- 用戶無永豐投顧帳號，方向 A（Cookie）無法執行
