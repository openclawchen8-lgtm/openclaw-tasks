# T007 - 抓取報告公開預覽內容（第一段摘要）

## 目標
實作 `fetch_report_preview(guid)` 函式，讀取報告詳情頁的公開預覽內容（第一段摘要）。

## 背景發現（2026-04-22）
- 報告詳情頁 `/Article/Inner/{guid}` 為公開可訪問（無需登入）
- 頁面內容結構：
  - **公開可見**：標題、日期、分類標籤、**第一段摘要內容**（~200-500字）
  - **需登入**：「登入會員，看更多」後的全部內容
- 示例（2026-04-22 上銀報告）：
  ```
  永豐投顧 SinoPac Inv.Service
  新聞評析｜Google新一代TPU即將亮相 聯發科衝漲停、市值3.3兆元寫新高
  2026.04.22
  ｜盤前必讀
  
  Google本周舉行Google Cloud Next'26大會，市場普遍預期，將端出最新一代 v8 架構張量處理器(TPU)，
  聯發科 (2454-TW) 因傳出拿下負責推論的TPU v8i 系列產品，
  今(21)日股價提前表態，衝上漲停板2,090元，市值也刷新紀錄，達新台幣3.3兆元。
  Wccftech日前報導，Google擬於本週推出新一代TPUv8 AI晶片...
  
  登入會員，看更多  ← 這裡之後是會員內容
  ```

## 待辦項目
- [☐] 實作 `fetch_report_preview(guid)` 函式
- [☐] 截取「登入會員，看更多」之前的文字
- [☐] 過濾導航列、頁尾、版權聲明等噪音元素
- [☐] 合併到 sinotrade_scraper.py
- [☐] 實測：6 篇報告都能抓到預覽內容
- [☐] 整合進 Telegram 通知（預覽內容作為摘要顯示）

## 實作思路

```python
async def fetch_report_preview(page, guid: str) -> str | None:
    """讀取報告公開預覽內容（第一段摘要）"""
    url = f"https://scm.sinotrade.com.tw/Article/Inner/{guid}"
    await page.goto(url, timeout=15000)
    await asyncio.sleep(2)

    # 取主要內容區
    body = await page.inner_text("body")

    # 截取「登入會員，看更多」之前的內容
    cut_marker = "登入會員，看更多"
    if cut_marker in body:
        content = body.split(cut_marker)[0]
    else:
        content = body  # 完全公開，無截斷

    # 過濾噪音
    noise = ["永豐投顧 SinoPac Inv.Service", "關於永豐投顧", "最新公告",
             "隱私權聲明", "客戶資料保密措施", "電話", "金管投顧"]
    for n in noise:
        content = content.replace(n, "")

    content = content.strip()
    return content if len(content) > 50 else None
```

4. Telegram 通知呈現正常
