## 完成摘要

### 已實現功能
1. **Dashboard 頁面**
   - 實時價格卡片（含今日高低）
   - K線圖展示（TradingView Lightweight Charts）
   - 最近決策列表
   - 5秒自動刷新

2. **Chart Analysis 頁面**
   - 4種時間框架（1H/4H/1D/1W）
   - 5種技術指標（MA/EMA/RSI/MACD/VOL）
   - RSI/MACD 數值面板
   - 走勢分析摘要

3. **Decision Detail 頁面**
   - 信號強度儀表
   - 買入/賣出/持有信號列表
   - 決策歷史表格

4. **Auth Page**
   - 登入/註冊表單
   - 表單驗證（格式、非空、密碼匹配）
   - 忘記密碼流程
   - 載入狀態處理

5. **圖表交互**
   - K線 + 成交量 + MA20
   - 十字準線
   - 自適應寬度
   - 深色主題

### 路由配置
```
/auth      → AuthPage
/          → Dashboard
/chart     → ChartAnalysis
/analysis  → ChartAnalysis
/history   → DecisionDetail
```
