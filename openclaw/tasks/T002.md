## 描述
How to install openclaw with npm under macbook when qclaw running.

## 安裝方式

### 方式一：全局安裝（推薦）
```bash
npm install -g openclaw
```

### 方式二：本地安裝
```bash
mkdir openclaw-project
cd openclaw-project
npm init -y
npm install openclaw
```

### 當前版本
- npm: 11.12.1
- OpenClaw: 2026.4.2 (d74a122)

### 驗證安裝
```bash
# 檢查 CLI
openclaw --version

# 檢查 Gateway 狀態
openclaw status

# 查看幫助
openclaw --help
```

### 常用命令
```bash
openclaw gateway start    # 啟動 Gateway
openclaw gateway stop     # 停止 Gateway
openclaw gateway status   # 查看狀態
openclaw gateway install  # 安裝為系統服務
```
