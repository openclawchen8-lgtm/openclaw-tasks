## Skill 安裝說明

### 安裝方式

#### 線上安裝（需要 Gateway 運行）
```bash
# 使用 SkillHub 安裝
openclaw skills install <skill-name>

# 使用 ClawHub 安裝
clawhub install <owner/skill-name>
```

#### 離線安裝（無需啟動 OpenClaw）
```bash
# 方法1：複製 Skill 目錄
cp -r <skill-source> ~/Library/Application\ Support/QClaw/openclaw/config/skills/<skill-name>/

# 方法2：使用 npm 安裝全球可用的 Skill
npm install -g <skill-npm-package>
```

### 驗證
```bash
# 查看已安裝 Skills
openclaw skills list
```
