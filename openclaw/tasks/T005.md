## 描述
How to create pre-defined members with agents's defined md files without startup openclaw.

### 離線創建成員

#### 目錄結構
```
~/.openclaw/agents/<agent-name>/
├── agent/           # Agent 配置
│   ├── IDENTITY.md  # 身份定義
│   ├── SOUL.md      # 個性定義
│   └── MEMORY.md    # 長期記憶（可選）
└── sessions/        # Session 存儲（自動創建）
```

#### 步驟
```bash
# 1. 建立目錄
mkdir -p ~/.openclaw/agents/<agent-name>/agent
mkdir -p ~/.openclaw/agents/<agent-name>/sessions

# 2. 創建 IDENTITY.md
cat > ~/.openclaw/agents/<agent-name>/agent/IDENTITY.md << 'EOF'
# IDENTITY.md - Who Am I?

- **Name:** [名稱]
- **Creature:** [物種]
- **Vibe:** [風格]
- **Emoji:** [表情]
EOF

# 3. 創建 SOUL.md
cat > ~/.openclaw/agents/<agent-name>/agent/SOUL.md << 'EOF'
# SOUL.md - Who You Are

## Core Truths
[核心特質描述]

## Boundaries
[邊界規則]

## Vibe
[溝通風格]
EOF

# 4. 權限設置
chmod -R 700 ~/.openclaw/agents/<agent-name>/
```

#### 啟動後註冊
```bash
# 重啟 Gateway 後自動發現新成員
openclaw gateway restart

# 或手動刷新
openclaw config reload
```

### 驗證
```bash
# 查看已註冊 Agents
openclaw agents list
```
