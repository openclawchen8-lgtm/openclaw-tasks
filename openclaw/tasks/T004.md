# T004 - How to install pre-defined openclaw plugins withou

## 基本資訊
- **Type**: skill
- **Assignee**: 安安
- **Priority**: medium

## 描述
How to install pre-defined openclaw plugins without startup openclaw.

## Plugin 安裝說明

### 預設 Plugins（12個）
位置：`~/Library/Application Support/QClaw/openclaw/config/extensions/`

```
dingtalk-connector      # 釘釘連接器
lossless-claw          # 無損壓縮
memory-tencentdb       # 騰訊資料庫記憶
openclaw-mem0          # mem0 記憶
openclaw-qqbot         # QQ 機器人
qclaw-plugin           # QClaw 核心插件
queue-guard            # 隊列守衛
skill-interceptor      # Skill 攔截器
wechat-access          # 微信通路
wecom-openclaw-plugin  # 企業微信
weixin                 # 微信
```

### 安裝方式

#### 自動載入（推薦）
Plugins 位於配置目錄時會自動載入，無需手動安裝：
```bash
# 路徑
~/Library/Application\ Support/QClaw/openclaw/config/extensions/<plugin-name>/
```

#### 手動安裝
```bash
# 複製插件目錄到擴展目錄
cp -r <plugin-source> ~/Library/Application\ Support/QClaw/openclaw/config/extensions/<plugin-name>/
```

#### npm 安裝
```bash
# 安裝全局 npm 包
npm install -g <plugin-npm-package>

# 或本地專案
npm install <plugin-npm-package>
```

### 驗證
```bash
# 啟動後查看 Plugin 狀態
openclaw status | grep -i plugin
```
