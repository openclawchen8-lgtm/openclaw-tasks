# T006 - OpenClaw Cron 備份與還原

## 基本資訊
- **Type**: Howto
- **Assignee**: 安安
- **Priority**: medium
- **Status**: done

## 描述
How to backup and restore for openclaw's cron tasks.

---

## 一、備份 Cron 任務

### 1.1 查看所有 Cron 任務

使用 Gateway API 或 CLI：

```bash
# 查看所有 cron 任務（通過 Gateway API）
curl -s -H "Authorization: Bearer <token>" \
  http://localhost:28789/api/cron/list

# 或在 OpenClaw Control UI > Schedule 查看
```

### 1.2 匯出 Cron 任務配置

用 `bash <skill_dir>/scripts/openclaw-mac.sh cron list` 列出所有任務 ID 和配置：

```bash
SKILL_DIR="~/Library/Application Support/QClaw/openclaw/config/skills/qclaw-openclaw"
bash "$SKILL_DIR/scripts/openclaw-mac.sh" cron list
```

### 1.3 備份到 JSON 檔

```bash
SKILL_DIR="~/Library/Application Support/QClaw/openclaw/config/skills/qclaw-openclaw"
BACKUP_DIR="$HOME/.qclaw/backups"
mkdir -p "$BACKUP_DIR"

# 擷取 cron 清單
bash "$SKILL_DIR/scripts/openclaw-mac.sh" cron list > "$BACKUP_DIR/cron-list-$(date +%Y%m%d_%H%M%S).txt"

# 或直接查詢並格式化
curl -s -H "Authorization: Bearer $(cat ~/.qclaw/openclaw.json | grep token | head -1)" \
  http://localhost:28789/api/cron/jobs \
  | python3 -m json.tool > "$BACKUP_DIR/cron-jobs-$(date +%Y%m%d_%H%M%S).json"
```

### 1.4 自動排程備份（每小時）

```json
{
  "action": "add",
  "job": {
    "name": "OpenClaw Cron 備份",
    "schedule": {"kind": "every", "everyMs": 3600000},
    "sessionTarget": "isolated",
    "payload": {
      "kind": "agentTurn",
      "message": "直接輸出以下提醒內容，禁止呼叫 message 工具：\n=== OpenClaw Cron 備份提醒 ===\n請執行備份腳本將 cron 任務匯出至 ~/.qclaw/backups/"
    },
    "delivery": {"mode": "announce"}
  }
}
```

---

## 二、還原 Cron 任務

### 2.1 從 JSON 還原

若備份了完整 cron job 配置（包含 id、name、schedule、payload、delivery），可直接用 `cron add` 逐個重建：

```bash
# 讀取備份檔
BACKUP_FILE="$HOME/.qclaw/backups/cron-jobs-YYYYMMDD_HHMMSS.json"

# 解析並重建（需逐筆執行）
# 格式：bash openclaw-mac.sh cron add --name "<name>" --every 30m ...
```

### 2.2 手動重建常見任務

```bash
# 黃金監控（每10分鐘，營業日）
bash <skill_dir>/scripts/openclaw-mac.sh cron add \
  --name "黃金存摺價格監控" \
  --cron "*/10 9-15 * * 1-5" --tz "Asia/Taipei"

# 每日收盤報告（15:30）
bash <skill_dir>/scripts/openclaw-mac.sh cron add \
  --name "黃金每日收盤報告" \
  --cron "30 15 * * 1-5" --tz "Asia/Taipei"

# Ideas 每日掃描（09:00）
bash <skill_dir>/scripts/openclaw-mac.sh cron add \
  --name "ideas2tasks 每日掃描" \
  --cron "0 9 * * *" --tz "Asia/Taipei"
```

## 三、驗證還原結果

```bash
# 確認任務數量一致
bash <skill_dir>/scripts/openclaw-mac.sh" cron list | wc -l

# 或查詢下一次執行時間
bash <skill_dir>/scripts/openclaw-mac.sh" cron list
```
