# T003 - 時間範圍配置

## 目標
實作彈性時間範圍參數，支援 3 種模式。

## 待辦項目
- [✅] 實作 `--range` CLI 參數（預設 `1d`）
  - `1d`：單日（預設，當日）
  - `3d`：近三日
  - `7d`：近一週
- [✅] 實作 `--date` 參數（指定特定日期，格式 YYYYMMDD）
- [✅] 在 `--range 3d` 或 `7d` 模式下，邏輯為「過去 N 天內有上傳」
- [✅] 整合 argparse 參數解析
- [✅] 輸出說明：`python3 kgi_monitor.py --help`

## 參數範例
```bash
python3 kgi_monitor.py                  # 今日（預設）
python3 kgi_monitor.py --range 3d       # 近三日
python3 kgi_monitor.py --range 7d        # 近一週
python3 kgi_monitor.py --range 1d --date 20260422  # 指定日期
```

## 驗證方式
對同一組影片，分別用 `--range 1d`、`3d`、`7d` 執行，確認結果數量遞增或一致。

## 備註
- yt-dlp 的 `--dateafter` 參數可用於過濾上傳日期
- 需處理時區問題（YouTube 為 UTC，轉換為台灣時間 UTC+8）
