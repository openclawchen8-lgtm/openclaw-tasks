# T005 - Cron 定時執行設定

## 目標
設定每日 cron job，21:30 自動執行（節目首播後 30 分鐘）。

## 待辦項目
- [✅] 設定 cron：每個交易日 21:30 執行
- [✅] 使用 systemEvent + 腳本自己發 Telegram（零 AI inference）
- [✅] Payload：`python3 /Users/claw/scripts/kgi_monitor.py --range 1d --telegram`
- [✅] sessionKey：`agent:main:cron:kgi-monitor`
- [✅] 設定完成後通知豪確認

## Cron 規格
```json
{
  "schedule": { "kind": "cron", "expr": "30 21 * * 1-5", "tz": "Asia/Taipei" },
  "payload": { "kind": "systemEvent", "text": "python3 /Users/claw/scripts/kgi_monitor.py --range 1d --telegram" },
  "sessionTarget": "main"
}
```

## 驗證方式
手動觸發一次 cron，確認收到 Telegram 通知。

## 備註
- 週一~週五執行（排除週末）
- 腳本自行發送 Telegram，不依賴 delivery
