# T001 - 環境建置：安裝 yt-dlp

## 目標
在主機上安裝 yt-dlp（YouTube 影片下載/資訊取得 CLI）。

## 待辦項目
- [✅] 確認是否已安裝 yt-dlp（`which yt-dlp`）
- [✅] 若無，安裝：`pip3 install yt-dlp` 或 `brew install yt-dlp`
- [✅] 測試取得頻道影片清單
- [✅] 驗證指令：`yt-dlp --flat-playlist --print "%(id)s|%(title)s|%(upload_date)s" "https://www.youtube.com/@KGISIA.channel/videos" --playlist-end 5`

## 驗證方式
指令成功輸出最近 5 筆影片的 ID、標題、上傳日期。

## 備註
- yt-dlp 不需要登入即可取得公開影片清單
- 可用 `--dateafter today` 過濾日期範圍
