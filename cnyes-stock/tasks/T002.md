# T002 - 核心功能開發 - 新聞列表抓取與解析

## 目標
開發 Playwright 腳本，抓取鉅亨網台股新聞列表與解析。

## 背景發現（2026-04-22 實測）

### URL 格式
```
https://news.cnyes.com/news/cat/tw_stock?date=YYYY-MM-DD
```
- `?date=2026-04-22`：過濾為指定日期的新聞 ✅ 有效

### 文章連結格式
```
/news/id/{numeric_id}
完整 URL: https://news.cnyes.com/news/id/{numeric_id}
```

### 文章 DOM 結構（典型）
```
<a href="/news/id/6429304">
  <分類標籤> 台股
  <時間> 17:05
  <標題> 盤後速報 - 長廣(7795)次交易(23)日除息2.15元，參考價418.85元
</a>
```

### 分類過濾
- 頁面包含多分類新聞，需過濾只取「台股」
- 可從 DOM 文字中用正規表達式抓 `^(台股)$`

### 滾動載入（重要）
- 頁面為無盡滾動（Infinite Scroll）
- 需多次滾動觸發載入，確保完整列表
- 建議：滾動到無新內容為止，或固定滾動 5-10 次

## 實作思路

```python
async def fetch_today_news():
    today = datetime.now().strftime("%Y-%m-%d")
    url = f"https://news.cnyes.com/news/cat/tw_stock?date={today}"

    await page.goto(url, timeout=15000)

    # 滾動觸發載入（無盡滾動）
    prev_count = 0
    for _ in range(10):
        await page.evaluate("window.scrollTo(0, document.body.scrollHeight)")
        await asyncio.sleep(1)
        curr_count = await page.locator("a[href*='/news/id/']").count()
        if curr_count == prev_count:
            break
        prev_count = curr_count

    # 解析新聞
    articles = []
    for a in await page.locator("a[href*='/news/id/']").all():
        text = await a.inner_text()
        if not re.search(r'^台股$', text, re.MULTILINE):
            continue  # 只取台股分類

        # 提取時間、標題
        match = re.search(r'(\d{2}:\d{2})\s*(.+)', text)
        time_str = match.group(1) if match else ""
        title = match.group(2).strip() if match else text.strip()

        articles.append({
            'url': 'https://news.cnyes.com' + await a.get_attribute("href"),
            'time': time_str,
            'title': title,
        })

    return articles
```
