# T003 - 增量比對邏輯

## 目標
實作增量比對：只通知當日新增的新聞，避免重複轟炸。

## 實作思路

完全參照 sinotrade-scraper 的增量邏輯。

### 歷史記錄格式
```json
// ~/.qclaw/cnyes_stock_history.json
{
  "2026-04-22": [
    {
      "time": "17:05",
      "title": "盤後速報 - 長廣(7795)...",
      "url": "https://news.cnyes.com/news/id/6429304"
    }
  ]
}
```

### 比對流程
1. 抓取今日新聞列表
2. 讀取 `cnyes_stock_history.json`
3. 比對：今日列表 vs 歷史記錄
4. 只取 `url` 不在歷史中的新聞作為新增
5. 寫入歷史記錄

### 檔案操作
```python
import json, pathlib

HISTORY_FILE = pathlib.Path.home() / ".qclaw/cnyes_stock_history.json"

def load_history():
    if HISTORY_FILE.exists():
        return json.loads(HISTORY_FILE.read_text())
    return {}

def save_history(date, news_list):
    hist = load_history()
    hist[date] = news_list
    HISTORY_FILE.write_text(json.dumps(hist, ensure_ascii=False, indent=2))

def get_new_news(date, news_list):
    hist = load_history()
    existing = set(n['url'] for n in hist.get(date, []))
    return [n for n in news_list if n['url'] not in existing]
```
