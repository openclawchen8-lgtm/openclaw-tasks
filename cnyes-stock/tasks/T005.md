# T005 - Cron 定時執行

## 目標
設定 Cron Job，每週一～五 08:30 自動執行 cnyes_stock_scraper.py。

## 排程設計
- **時間**：每週一～五 08:30（Asia/Taipei）
- **sessionKey**：`agent:main:cron:cnyes-stock`
- **payload**：systemEvent（純腳本，零 AI inference）
- **通知**：腳本內自幹 `send_telegram`（不依賴 delivery）

## Cron Job 設定
```python
{
  "name": "cnyes-stock 每日通知",
  "schedule": {
    "kind": "cron",
    "expr": "30 8 * * 1-5",
    "tz": "Asia/Taipei"
  },
  "payload": {
    "kind": "systemEvent",
    "text": "python3 /Users/claw/scripts/cnyes_stock_scraper.py --telegram"
  },
  "sessionTarget": "main",
  "sessionKey": "agent:main:cron:cnyes-stock",
  "enabled": true
}
```
