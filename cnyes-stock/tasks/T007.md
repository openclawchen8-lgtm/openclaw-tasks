# T007 - 新增 Trending 頁面台股區塊抓取

## 實作方案

### 方案 A：修改現有 scraper 新增參數
在 `cnyes_stock_scraper.py` 新增 `--trending` 參數：
```bash
python3 cnyes_stock_scraper.py --trending          # 抓取 trending 台股區塊
python3 cnyes_stock_scraper.py --trending --telegram
```

### 方案 B：獨立 script
建立 `cnyes_trending_scraper.py` 專門處理 trending 頁面。

**建議：方案 A**，統一管理較方便。

## 技術實作

### 抓取邏輯
1. 開啟 `https://news.cnyes.com/trending?exp=a`
2. 等待頁面載入（networkidle）
3. 定位「台股」區塊（heading "台股"）
4. 抓取該區塊內的 10 個連結（`a[href*='/news/id/']`）
5. 解析：編號、標題、URL

### 輸出格式
```json
{
  "source": "trending",
  "section": "台股",
  "timestamp": "2026-04-23T15:30:00",
  "news_count": 10,
  "news": [
    {
      "rank": 1,
      "title": "估值要重來了！聯發科升級AI供應商、鴻海毛利結構翻轉",
      "url": "https://news.cnyes.com/news/id/xxxxxxx"
    }
  ]
}
```
