## 發布平台評估

### Gumroad

| 優點 | 缺點 |
|------|------|
| 開箱即用，5 分鐘上架 | 抽成 10% |
| 支援全球付款（信用卡、PayPal）| 無中文客服 |
| 自動發送下載連結 | 課程功能弱（只能放影片連結）|
| 有 Pro 會員市場曝光 | 需要自己宣傳 |

**適合**：快速上架、不想花時間搞網站的人

### Hahow

| 優點 | 缺點 |
|------|------|
| 中文最大學習平台，流量大 | 審核嚴格，需試教影片 |
| 支援分期付款 | 抽成較高（營收 30%+）|
| 課程功能完整（章節、測驗）| 必須配合平台行銷活動 |
| 中文客服 | 無法直接聯繫學員（平台掌握）|

**適合**：想服務中文市場、願意配合平台節奏

### 自建網站（Ghost / WordPress）

| 優點 | 缺點 |
|------|------|
| 完全掌控，無抽成 | 需要技術能力或花錢請人 |
| 可整合會員系統、論壇 | 需要自己處理金流（Stripe/LINE Pay）|
| 建立自有品牌 | 行銷從零開始 |

**適合**：長期經營、已有流量來源的人

### 我的建議

| 階段 | 平台 |
|------|------|
| 第一批（測試水溫）| Gumroad — 快、簡單、先賣再說 |
| 第二批（建立品牌）| 自建站（Ghost）— 長期客戶關係 |
| 第三批（擴大曝光）| Hahow — 有名氣後進駐 |

**起步策略**：先用 Gumroad 上架教程 1，同時經營 GitHub + Medium + Telegram 建立流量來源，再考慮自建站。
