# 🤖 AI 自動化監控系統

**用自然語言驅動的智能監控與自動化系統**

從黃金價格追蹤到全自動化工作流，零成本打造專屬 AI 助理。適合個人投資者、技術人與小型團隊。

---

## 📌 產品簡介

AI 自動化監控系統是一套基於 OpenClaw Agent Framework 打造的智能監控解決方案。**無需程式背景**，用自然語言即可定義監控任務；**全自動運行**，24/7 不間斷守護你的投資與數據。

**旗艦產品 Gold Monitor Pro** 專為台灣黃金投資者設計，即時追蹤台灣銀行黃金存摺牌告價，支援買入/賣出雙價格顯示、價格變動告警、每日收盤報告等完整功能。進階版更支援白銀、鉑金等多金屬監控，以及台銀與國際現貨價對比分析。

對於技術用戶，本系統提供完整的 OpenClaw Skill 擴展能力——透過 Python 腳本、Cron 定時任務、Telegram Bot API 整合，你可以打造任何想像得到的自動化工作流。從簡單的價格通知到複雜的多雲端備份，一切盡在掌控。

## 🏗️ 技術架構

本系統採用模組化設計，核心架構如下：

```
┌─────────────────────────────────────────────────────────┐
│                  OpenClaw Agent Framework               │
│                    (自然語言驅動核心)                     │
└─────────────────────────────────────────────────────────┘
                           │
         ┌─────────────────┼─────────────────┐
         ▼                 ▼                 ▼
   ┌──────────┐      ┌──────────┐      ┌──────────┐
   │  Python  │      │ Telegram │      │  SQLite  │
   │  Scripts │      │ Bot API  │      │ 本地數據庫│
   └──────────┘      └──────────┘      └──────────┘
         │                 │                 │
         ▼                 ▼                 ▼
   ┌──────────┐      ┌──────────┐      ┌──────────┐
   │  Cron    │      │  價格    │      │  GitHub  │
   │  定時器   │      │  通知    │      │  Actions │
   └──────────┘      └──────────┘      └──────────┘
```

**核心元件說明：**

- **OpenClaw Agent Framework** — 自然語言處理引擎，讓你用中文定義監控任務
- **Python Scripts** — 數據抓取、分析、處理的核心邏輯
- **Telegram Bot API** — 即時推送價格告警與每日報告
- **SQLite** — 輕量級本地數據庫，儲存歷史價格與配置
- **Cron Jobs** — 精準定時執行，確保不漏任何價格變動
- **GitHub Actions** — 自動備份腳本與數據到雲端

**開源社群：** 所有核心腳本開源於 [GitHub: openclawchen8-lgtm](https://github.com/openclawchen8-lgtm)，歡迎貢獻與客製化。

## 🚀 快速開始

### 前置需求
- 已安裝 OpenClaw（[安裝指南](https://openclaw.ai)）
- Telegram 帳號（用於接收通知）

### 步驟

1. **安裝 Skill**
   ```bash
   clawhub install ai-automation-monitoring
   ```

2. **設定 Telegram 通知**
   ```
   /設定 telegram_token YOUR_BOT_TOKEN
   /設定 telegram_chat_id YOUR_CHAT_ID
   ```

3. **啟動黃金監控**
   ```
   /啟動黃金監控
   ```

4. **設定價格告警**
   ```
   /設定告警 黃金 目標價 2500
   /設定告警 黃金 變動 1%
   ```

5. **查看狀態**
   ```
   /監控狀態
   ```

完成！系統將自動每10分鐘檢查價格，並在價格觸發條件時發送 Telegram 通知。
