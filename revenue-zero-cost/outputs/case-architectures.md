## 圖1：黃金監控系統（Gold Monitor Pro）

```
┌─────────────────────────────────────────────────────────┐
│                  Gold Monitor Pro 架構                   │
└─────────────────────────────────────────────────────────┘

  ┌──────────────┐     ┌──────────────────┐     ┌───────────┐
  │ 台灣銀行官網 │ ──→ │  Python 爬蟲     │ ──→ │  SQLite   │
  │ rate.bot.com│     │  (urllib/BS4)    │     │  price_db │
  └──────────────┘     └──────────────────┘     └─────┬─────┘
                                                      │
                          ┌───────────────────────────┤
                          │           分析引擎          │
                          │  • 門檻檢查（漲跌%/偏離均價） │
                          │  • 告警觸發判斷             │
                          └───────────────┬────────────┘
                                          │
              ┌───────────────────────────┼───────────────────────────┐
              │                           │                           │
              ▼                           ▼                           ▼
    ┌─────────────────┐        ┌──────────────────┐       ┌──────────────┐
    │   Telegram Bot  │        │    Email (SMTP)  │       │   每日收盤    │
    │   即時告警推送  │        │   失敗時通知      │       │   22:00報告  │
    └─────────────────┘        └──────────────────┘       └──────────────┘

排程：每 10 分鐘 (09:00-21:00)     每日 22:00
```

### 說明
- 資料來源為台灣銀行官網，無需 API Key
- SQLite 儲存所有歷史報價，無 local state 檔
- 告警門檻可自訂（漲跌 %、偏離均價 %）

## 圖2：備份自動化（OpenClaw Backup）

```
┌─────────────────────────────────────────────────────────┐
│              OpenClaw 備份自動化 流程                    │
└─────────────────────────────────────────────────────────┘

  ┌──────────────┐   每2小時   ┌──────────────────┐
  │   cron 觸發  │ ─────────→  │  backup.sh 腳本  │
  └──────────────┘             └────────┬─────────┘
                                       │
                    ┌──────────────────┼──────────────────┐
                    │                  │                  │
                    ▼                  ▼                  ▼
           ┌────────────┐     ┌───────────────┐    ┌─────────────┐
           │ Skills/    │     │  Howto/       │    │ Tasks/      │
           │ ~120 MB    │     │  Docs/        │    │ Projects/   │
           └────────────┘     └───────────────┘    └─────────────┘
                    │                  │                  │
                    └──────────────────┼──────────────────┘
                                       │
                            ┌──────────▼──────────┐
                            │   tar.gz 打包      │
                            │   (~130 MB)        │
                            └──────────┬──────────┘
                                       │
                    ┌──────────────────┴──────────────────┐
                    │                                     │
                    ▼                                     ▼
         ┌─────────────────────┐            ┌─────────────────────┐
         │  GitHub Releases    │            │   Telegram 通知      │
         │  (永久存檔)          │            │   ✅ 成功 / ❌ 失敗  │
         └─────────────────────┘            └─────────────────────┘
                    │
                    ▼
         ┌─────────────────────┐
         │  清理腳本            │
         │  刪除 >30天舊存檔   │
         └─────────────────────┘
```

### 說明
- 備份覆蓋：Skills、Howto、Tasks、Agents、Workspace
- 排除：`.git`、`node_modules`、Session logs、`.openclaw`
- GitHub Releases 作為永久存檔，任何時間可下載還原

## 圖3：ideas2tasks 工作流

```
┌─────────────────────────────────────────────────────────┐
│              ideas2tasks 自動化工作流                    │
└─────────────────────────────────────────────────────────┘

  ┌──────────────┐   每小時   ┌──────────────────┐
  │ /claw/Ideas/ │ ─────────→ │  lifecycle.py    │
  │  .txt 想法檔 │           │  掃描 + 分類     │
  └──────────────┘           └────────┬─────────┘
                                      │
                    ┌─────────────────┼─────────────────┐
                    │                 │                 │
                    ▼                 ▼                 ▼
           ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
           │ task.N 格式  │  │ [x] 核取方框  │  │ ## Todo      │
           │ 正規解析      │  │ 中文/英文解析  │  │ 多格式支援   │
           └──────────────┘  └──────────────┘  └──────────────┘
                    │                 │                 │
                    └─────────────────┼─────────────────┘
                                      │
                            ┌─────────▼────────────┐
                            │  三層防護去重引擎     │
                            │  1. 精確比對         │
                            │  2. 正規化比對        │
                            │  3. 相似度 >0.8       │
                            └─────────┬────────────┘
                                      │
                    ┌─────────────────┼─────────────────┐
                    │                 │                 │
                    ▼                 ▼                 ▼
         ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
         │ Tasks/*.md   │  │ GitHub Issues│  │ GitHub Board │
         │ 任務事實來源  │  │  程式碼相關  │  │  進度視覺化  │
         └──────────────┘  └──────────────┘  └──────────────┘
                    │
                    ▼
         ┌──────────────────────┐
         │  state_sync.py        │
         │  三方狀態同步鉤子      │
         └──────────────────────┘
                    │
                    ▼
         ┌──────────────────────┐
         │  所有 tasks 完成 →    │
         │  自動歸檔 _done/     │
         └──────────────────────┘
```

### 說明
- Tasks/ 目錄狀態為唯一事實來源
- 去重三層防護：精確 → 正規化 → 相似度
- 所有完成的想法自動移入 `_done/` 目錄
