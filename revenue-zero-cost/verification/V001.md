## V002 新發現問題

### Alpha Vantage API 函數名稱錯誤【高優先】

**檔案：** `scripts/data_adapters/alpha_vantage_adapter.py`

**問題：** 使用 `function: "GOLD"` 和 `function: "SILVER"` 但 API 回應錯誤：
> "This API function (GOLD) does not exist."

**原因：** Alpha Vantage 免費方案不支援這些 commodity APIs，可能需要使用 `CURRENCY_EXCHANGE_RATE` 搭配商品貨幣代碼 (如 XAU/USD) 或切換到付費方案。

**修復方向：**
1. 測試使用 `function: "CURRENCY_EXCHANGE_RATE"` 搭配 `from_currency: "XAU"` (黃金)
2. 或查詢 Alpha Vantage 官方文檔確認正確的 function 名稱
