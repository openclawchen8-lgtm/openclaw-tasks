# T006 - 製作案例作品集

## 目標
整理並展示已完成的自動化專案，作為服務銷售的案例

## 案例項目

### 1. 黃金監控系統
- **技術棧**：Python, Playwright, Telegram API, FastAPI
- **功能**：多金屬監控、國際價對比、多通道告警
- **價值**：投資決策輔助，省去手動查價

### 2. 備份自動化
- **技術棧**：Bash, cron, GitHub, 雲端存儲
- **功能**：定時備份、多雲同步、容災恢復
- **價值**：資料安全，零人為干預

### 3. ideas2tasks 工作流
- **技術棧**：OpenClaw, YAML, Git
- **功能**：想法掃描、專案建立、任務拆解
- **價值**：創意管理自動化
