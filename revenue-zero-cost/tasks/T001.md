# T001 - 升級黃金監控為付費版

## 目標
將 free-gold-monitor 升級為付費版本，增加價值功能

## 升級功能規劃
| 功能 | 免費版 | 付費版 |
|------|--------|--------|
| 金屬種類 | 黃金 | 黃金 + 白銀 + 鉑金 |
| 數據源 | 台銀單一源 | 台銀 + 國際價對比 |
| 歷史數據 | 7 天 | 1 年 |
| 告警通道 | Telegram | Telegram + Email + Webhook |
| 圖表 | 基礎線圖 | TradingView 完整圖表 |
| API | 無 | REST API |

## 技術要點
- 擴展 `gold_monitor.py` 支持多金屬
- 增加 Alpha Vantage / Finnhub 國際價數據源（複用 gold-analysis 的 data adapters）
- 歷史數據存 SQLite（已有基礎）
- 告警通道增加 Email（複用 imap-smtp-email）
