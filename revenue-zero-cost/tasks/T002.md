# T002 - 設計付費方案

## 目標
設計 ClawHub 技能的付費方案與定價策略

## 方案設計

### 免費版（引流）
- 黃金單一監控
- Telegram 告警
- 7 天歷史數據
- 社群支持

### 進階版 - $9.9/月 或 $99/年
- 多金屬監控（黃金/白銀/鉑金）
- 國際價對比
- 1 年歷史數據
- 多通道告警（Telegram + Email）
- Email 技術支持

### 企業版 - $299/年
- 全部進階功能
- REST API 訪問
- 自定義數據源
- 優先技術支持
- 私有部署諮詢

## 定價邏輯
- 免費版引流，展示能力
- 進階版對標其他監控服務（便宜 50%+）
- 企業版高毛利，服務 B 端

## 目標轉換率
- 免費 → 進階：5%
- 進階 → 企業：10%
